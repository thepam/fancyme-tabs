The main HTML structure needed to build this tab is pretty simple, described on the following block of code.

<div id="mytabs">
    <ul>
        <li class="current">Tab 1</li>
        <li>Tab 2</li>
        <li>Tab 3</li>
        <li>Tab 4</li>
    </ul> 
    <div>
        <p>...Some content...</p>
    </div>
    <div>
        <p>...Some content...</p>
    </div>
    <div>
        <p>...Some content...</p>
    </div>
    <div>
        <p>...Some content...</p>
    </div>
</div>

With the necessary imports in the HEAD of your page

<link rel="stylesheet" type="text/css" href="fancymetabs.css"/>
<script type="text/javascript" src="jquery-1.9.0.min.js"></script>
<script type="text/javascript" src="fancymetabs.jQuery.js"></script>

And the JQuery function call, also in the HEAD

<script>
$(function() {
    $('#mytabs').fancymetabs();
});
</script>

As any FancyMe plugin does, the tab menu can be customized with predetermined color themes like it's shown in the following function call.

<script>
$(function() {
    $('#mytabs').fancymetabs({theme: "blue"});
});
</script>

The available themes are:

  • default (gray tones)
  • blue
  • red
  • pink
  • green
  • yellow
  • orange
  • purple
  • black

Another customizable preference is the height of the tabs. If left default, height varies according to the content, as seen in the previous examples. In this example, thought, the height is fixed through the following code.

<script>
$(function() {
    $('#mytabs').fancymetabs({theme: "purple", tabheight: "350px"});
});
</script>

The parameter tabheight must receive a value in pixels as the code shows, and that will mark the fixed height for all of the tabs. If the parameter isn't specified, the height will vary with the content.
